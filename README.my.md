**BETA — အလယ်ပိုင်း-အယ်လ်ဖာ။ Tor-DNS-သိရှိသော OS တွင် ပေါက်ပြဲမှု-ခိုင်ခံ့။ မစတင်မီ [ကြိုတင်လိုအပ်ချက်များ](README.md#before-you-install) ကို ဖတ်ပါ။**


OnionBird သည် IMAP/SMTP ကို ဒေသခံ Tor ပရောက်စီမှ ဖြတ်သန်းပြီး စာပို့သူများ၏ အမည်ဝှက်ဖျက်ပစ်ရန် သမိုင်းကြောင်းအရ အသုံးပြုခဲ့သော mail header များကို ဖယ်ရှား သို့မဟုတ် ပုံမှန်ပြုသော Thunderbird add-on တစ်ခုဖြစ်သည်။ Thunderbird 128+ ကို ဦးတည်သည် (ပင်မရည်ရွယ်ချက် TB 140 ESR)။ စွန့်ပစ်ထားသော TorBirdy extension ၏ ခေတ်မီအဆက်အနွယ်အဖြစ် ဒီဇိုင်းရေးထားသည် (နောက်ဆုံးထုတ်လုပ်မှု v0.2.6 ၂၀၁၈ ခုနှစ်တွင်; Mozilla က TB 78 တွင် Legacy XUL ဖယ်ရှားသောအခါ သေဆုံးခဲ့သည်)။

> **မှတ်ချက်:** ဤသည် AI-ဘာသာပြန်ဆိုထားသော ဗားရှင်းတစ်ခုဖြစ်သည်။ မိခင်ဘာသာစကားပြောသူများ၏ ပါဝင်မှု (idiom၊ လုံခြုံရေးဆိုင်ရာ ဝေါဟာရ၊ ပြန်ပြုပြင်ခြင်း) ကို PR ဖြင့် ကြိုဆိုသည်။
