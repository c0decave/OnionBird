**ALPHA — كېيىنكى-ئالفا. Tor-DNS-بىلىدىغان OS دە سىزىش-مۇستەھكەم. ئورنىتىشتىن ئىلگىرى [ئالدىنقى شەرتلەرنى](README.md#before-you-install) ئوقۇڭ.**


OnionBird بولسا Thunderbird قىستۇرمىسى بولۇپ، IMAP/SMTP نى يەرلىك Tor proxy ئارقىلىق يۆتكەيدۇ ۋە ئەۋەتكۈچىلەرنىڭ كىملىكىنى يوقىتىش ئۈچۈن تارىختا ئىشلىتىلگەن mail headers لارنى ئېلىپ تاشلايدۇ ياكى نورماللاشتۇرىدۇ. نىشانى Thunderbird 128+ (ئاساسلىق نىشان TB 140 ESR). تاشلىۋېتىلگەن TorBirdy كېڭەيتمىسىنىڭ زامانىۋى ئورۇنباسارى سۈپىتىدە لايىھەلەنگەن (ئەڭ ئاخىرقى نەشرى v0.2.6 2018-يىلى؛ Mozilla TB 78 دە Legacy XUL نى ئېلىۋەتكەندە ئۆلدى).

> **ئەسكەرتىش:** بۇ AI-تەرجىمە قىلىنغان نۇسخا. ئانا تىلدا سۆزلىگۈچىلەرنىڭ تۆھپىسى (idiom، بىخەتەرلىك ئاتالغۇلىرى، تۈزىتىش) PR ئارقىلىق قارشى ئېلىنىدۇ.
