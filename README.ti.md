**ALPHA — ድሕረ-ኣልፋ። ኣብ Tor-DNS-ዝፈልጥ OS ጹሕ-ጽኑዕ። ቅድሚ ምስፋር [ቅድመ-ኩነታት](README.md#before-you-install) ኣንብብ።**


OnionBird ናይ Thunderbird ተወሳኺ ኮይኑ ንIMAP/SMTP ብናይ ዓዲ Tor proxy ዘመሓላልፍ እዩ፣ ናይ ላኣኽቲ መንነት ምውጻእ ኣብ ታሪኽ ዝተጠቐመ ናይ ኢመይል headers ይኣልዮም ወይ ይምድብ። ናብ Thunderbird 128+ ይዓልም (ቀንዲ ዕላማ TB 140 ESR)። ኣብቲ ዝተገድፈ TorBirdy ቅጥያ ናይ ዘመናዊ ተካኢ ኮይኑ ዝተሰርሐ (ናይ መወዳእታ release v0.2.6 ኣብ 2018፤ Mozilla ናይ Legacy XUL ኣብ TB 78 ምስ ኣወገደ ሞተ)።

> **ኣስተውዕል:** እዚ AI-ዝተተርጎመ ቅዳሕ እዩ። ናይ ተወላዲ ተዛራቢ ኣበርክቶ (idiom፣ ናይ ድሕነት ቃላት፣ ምትዕርራይ) ብPR ይተባባዕ።
