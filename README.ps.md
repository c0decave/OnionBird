OnionBird د Thunderbird یوه اضافه ده چې IMAP/SMTP د محلي Tor proxy له لارې مسیر کوي او د لیږونکو پیژندنه له منځه وړلو لپاره په تاریخي ډول کارول شوي د میل headers لرې کوي یا نورمالوي. هدف Thunderbird 128+ (لومړنی هدف TB 140 ESR). د پریښودل شوي TorBirdy غزونې عصري ځای ناستی په توګه ډیزاین شوی (وروستی نسخه v2018 v0.2.6؛ کله چې Mozilla په TB 78 کې Legacy XUL لرې کړل، ووژل شو).
