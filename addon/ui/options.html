Message-ID FQDN

      

        The FQDN in outgoing Message-IDs is a recurring fingerprint
        across your mail. Default uses your From-address domain so you
        blend with regular users at the same provider. TorBirdy's
        localhost.localdomain mode remains available as an explicit
        legacy choice.
      

      

        
        
        
      

      
 

    


    

      
SOCKS endpoint override

      

        By default the addon detects Thunderbird's existing
        network.proxy.socks pref or falls back to 127.0.0.1 on port
        9050 / 9150. Set host + port here to pin a different endpoint
        (e.g. a Whonix Gateway IP). Host must be a loopback name
        (`localhost`) or an IP literal — DNS-resolvable names would
        leak via the system resolver before Tor.
